## Deskripsi singkat aplikasi
Aplikasi "PokéCare" sederhana berbasis PHP native yang mensimulasikan pelatihan Pokémon (Ivysaur). Menggunakan konsep OOP (Abstraction, Encapsulation, Inheritance, Polymorphism). Fitur:
- Halaman Beranda: info dasar Pokemon
- Halaman Latihan: form latihan, memanggil method `train()` dan `specialMove()`
- Halaman Riwayat Latihan: menyimpan dan menampilkan sesi latihan dalam `data/history.json`
